### 数据结构

#### 树状数组

> 用法在于单点更新，区间查询，这里的单点更新是单点加减，换的话好像不是不行
这个算法是利用了一个二进制的操作，只要记得要初始化就行。
```cpp
int tree[maxn]
int lowbit(int x)
{
    return x&(-x);
}
void add(int st,int flag) // st这个位置加flag的值
{
    for(int i = st;i<=n;i+=lowbit(i))
    {
        tree[i] += flag;
    }
}
int getsum(int st)  // 前st的数的和，包括st
{
    int res = 0;
    for(int i = st;i>0;i-=lowbit(i))
    {
        res += tree[i];
    }
    return res;
}
```
